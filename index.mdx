## What NQ does

NQ pulls together your activity from the services you already use — what you've watched, played, listened to, and read — and builds a graph of your media preferences. From that graph, it calculates personalized recommendations for what to consume next.

## Get started in three steps

<Steps>
  <Step title="Install prerequisites">
    Install Go 1.19+, Node.js 18+, and set up a Neo4j database (free cloud or local Docker). See the [Installation guide](/installation).
  </Step>
  <Step title="Start the backend and app">
    Run `go run .` in the `backend/` directory to start the GraphQL server, then `npx expo start` in `frontend/` to launch the mobile app. See the [Quick Start](/quickstart).
  </Step>
  <Step title="Connect your media services">
    Add API keys for Spotify, Steam, YouTube, and other services to your `.env` file. See [Connecting Services](/guides/connecting-services).
  </Step>
</Steps>

<Note>
  NQ is self-hosted — your media history and preferences never leave your own infrastructure. The only external calls are to the APIs you explicitly configure.
</Note>
